Python

准备从今天开始学习Python了，久仰大名准备一探究竟，废话不多说，let’s start

安装Python

第一步当然是安装Python 直接上 Python官网 download一下（注意环境）
下载好后直接默认安装，可以更改路径，此处就应该顺手把路径复制一下，方便等下配置环境变量（我发现我很喜欢纠结是否必须安装在C盘，可能是因为以前吃过亏hhh）
然后现在就可以配置环境变量了，找到path，编辑注意在末尾加上分号（英文的，当然）把你刚刚复制的安装路径粘贴进去就ok了
然后Win + R 输入cmd进入命令框 输入Python 看是否安装成功
